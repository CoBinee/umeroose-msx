## 説明
埋めてください。<br>
こちらから遊べます。⇒  [WebMSX](http://webmsx.org/?MACHINE=MSX1J&DISK=https://github.com/CoBinee/umeroose-msx/raw/main/diskimage/umeroose.dsk)

## 操作
- `SPACE`<br>押し続けると→に移動、離すと←に移動<br>離した瞬間に３マス先をブロックで埋める
- ジョイスティックにも対応

## 動作環境
- 32KB 以上の RAM を搭載した MSX
